# Proyek-Kolaborasi-Python---ALT-F4-
Game Tic Tac Toe

Program ini adalah permainan klasik Tic Tac Toe (X dan 0) yang berbasis pyhton dengan antarmuka pengguna grafis (GUI) menggunakan Tkinter. Dirancang untuk dua pemain, aplikasi ini melacak skor antar ronde, mengelola logika permainan (giliran, menang, seri), dan menyediakan antarmuka yang bersih serta tombol reset yang mudah digunakan.

# Cara Menjalankan Program
1. Pastikan python 3 sudah terinstal di device kalian
2. Download dua file utama (tictactoe_game.py dan gui_app.py) lalu menaruhnya di satu folder yang sama
3. Buka editor/interperter kalian dan jalankan kodenya lewat gui_app.py
